# 存链（tuku）变更日志

> 逐条细节看 git log；这里按批次记「做了什么、为什么」。

## 2026-07-12

### 一、品牌与形态
- 图床 → **存链**（品牌名，内部代号仍 tuku）；「图床-基础/专业」改名「存链-基础/专业」（app_key 不变，卡照用）。
- 第二期文件床：图片走 CF Images，其它文件（视频/音频/PDF/压缩包/任意）走 R2；配额从张数改**按容量 GB**。

### 二、客户端整体重构为 App 骨架（对标并超越 Lsky）
- 左侧总入口侧栏：仪表盘 / 上传 / 分类（全部/图片/视频/音频/文档/压缩/其他，带数量）/ 相册 / 设置。
- 仪表盘：大卡片（文件数/已用/可用/总量）+ 容量进度条 + 我的信息 + 最近上传。
- 我的文件 = 选择式图库：黄金比缩略图（名+时间）、多选批量条（移动/下载/删除）、⋯ 菜单（多格式复制/下载/新窗口/重命名/详情/移动/删除）、搜索排序、点图灯箱。
- 便捷：多格式复制（直链/MD/HTML/BBCode/缩略图）、Ctrl+V 粘贴上传、大文件断点续传、favicon。
- 手机端：☰ 抽屉侧栏 + 2 列网格 + 底部弹层 + 吸底批量条（响应式 @820px）。
- 运营台 /scfw（原 /admin，防猜）同款骨架：看板 + 客户管理（搜索/状态筛选/改档/停服/删除）。
- 内容区去 max-width 铺满宽屏（消除右侧大片留白）。

### 三、安全加固（第三方审查 + 验收两轮）
- 文件直链 IDOR：`/f/<数字id>` → `/f/<id>~<HMAC令牌>`，裸 id/错令牌拿不到。
- 存储型 XSS：上传 html/svg 强制下载 + CSP sandbox + nosniff；视频/PDF 保留内联。
- CSP 去 `unsafe-inline`：改逐请求 nonce（脚本仍内联）。
- 分片配额造假：complete 只认 `R2.head` 真实大小，拿不到即拒；普通上传并发 TOCTOU 兜底回滚。
- 登录：防账号枚举（未知卡/错密码/短密码统一 401；停用到期先验密码再暴露）；密码开通下限 8 位（不锁老用户）；暴破锁按 IP + 卡号双维度。
- 孤儿对账：上传写库失败回滚存储；删除失败入 `pending_deletes`，每 6h `scheduled` cron 补删（migration-v3）。
- 其它：密码 PBKDF2、上游 fetch 超时、esc 转义引号、schema.sql 合并为权威建表、verify.mjs 加「求值前端模板→node --check」。

### 四、获客与商业化
- **落地页**：未登录显示营销页（Hero + 功能卡 + 价格表 + 开发者亮点 + 页脚），登录改弹层。
- **免费档**：500MB + 图片水印 + 按 IP 限注册；`POST /api/free-signup` 自助开通 `FREE-xxxx`。
- **开发者 API**：`POST /api/v1/upload`（兰空/PicGo/Typora 兼容），设置里生成/复制/重置密钥（付费档专属）；migration-v4 加 `api_key` 列。
- **撞墙升级 / 到期提醒**：上传 402 弹升级窗、仪表盘顶部横幅，指向 `BUY_URL`（可配置）。
- **信任页** `/privacy`、`/terms`；运营台加获客指标（新增/付费转化/临期）。
